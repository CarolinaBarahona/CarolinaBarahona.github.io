COLE HARDWARE

GA Project | Solo project | 2 weeks

Cole Hardware is a retail hardware outlet, dealing with an extensive cross section of clients.
Cole Hardware store’s website has a category intended to give DIY enthusiast users some tips, but it doesn’t sell DIY products through the section so the user has to go to the shopping menu and then go to a different flow to finally find the products.
My solution was to add a feature to the menu that allows the user to go into an easier flow and to a faster “buying online” process. I also changed some labels (my extra gift to the users) that weren’t following conventions.
